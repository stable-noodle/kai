# 貝化装置 / 大航海時代オンライン クエストDB検索

## 概要

2つのツールを1ページにまとめたWebアプリです。

**貝化装置**
「かい」と読む漢字をすべて「貝」に置き換えます。形態素解析（kuromoji）による文脈判定モードと、単純置換モードを切り替えられます。

**クエストDB検索**
大航海時代オンラインのクエストデータをローカル検索します。クエスト名・必要スキル・受注都市・オファー方法などを横断的に絞り込み、発見物の詳細説明もトグルで確認できます。

## ファイル構成

```
kai/
├── index.html       # メインページ（全UI・ロジック）
├── quests.js        # クエストデータ（6,754件）
├── discovery.js     # 発見物データ（クエストIDで紐付け）
└── README.md
```

## 使い方

`index.html` をブラウザで開くだけで動作します。ローカルサーバーは不要です。

形態素解析（文脈判定モード）のみ、kuromoji 辞書の取得にインターネット接続が必要です。オフライン環境では自動的に単純置換モードで動作します。

## データ出典

クエスト・発見物データは以下のサイトの情報を元に作成しています。

- **大航海時代データベース** (gvdb.mydns.jp)

データの利用・再配布については元サイトの規約に従ってください。

## 技術スタック

- 純粋な HTML / CSS / JavaScript（フレームワーク・ビルドツールなし）
- 形態素解析：[kuromoji.js](https://github.com/takuyaa/kuromoji.js) v0.1.2（CDN経由、Web Worker内で動作）
- 辞書データ：jsdelivr CDN

## 注意事項

- kuromoji の辞書ダウンロード（初回）に数秒かかる場合があります。辞書が読み込まれるまでは自動的に単純置換モードで動作します。
- クエストデータは CSV からオフラインで変換したものです（データ最終更新：2026-06-23）。最新情報は元サイトをご確認ください。
